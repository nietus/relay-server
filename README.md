# RelayServer

## Local Development

```
cargo run
```

## Railway Deployment

To deploy this application to Railway:

1. Install the Railway CLI:

```
npm i -g @railway/cli
```

2. Login to Railway:

```
railway login
```

3. Link to your Railway project:

```
railway link
```

4. Deploy the application:

```
railway up
```

The application will be built using the Dockerfile in the repository.
